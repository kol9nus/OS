#Локер

###Использование

```
Программа просто дописывает в конец файла (этакий логер)
Использование: 
    locker <path> <message>[ <time>]
        ,где path - имя файла, в который будет записан message
             message - сообщение, которое будет записано в файл.
             time - сколько секунд длится запись, дефаут - 0
```
